- Accessibility standards compliant with WCAG 2.1 AA (`vitest-axe`, `eslint-plugin-jsx-a11y`).
- Rate limiting proxy-safe logic for Vercel deployment.
- High-level markdown documentation (`ARCHITECTURE.md`, `SECURITY.md`, etc.).
