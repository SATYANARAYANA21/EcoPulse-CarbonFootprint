# Contributing to EcoPulse

First off, thank you for considering contributing to EcoPulse!

## Development Setup
1. Clone the repo.
2. Ensure you have `Node.js >= 20` and `Python >= 3.11` installed.
3. For frontend, run `npm install` in `/frontend`.
4. For backend, run `pip install -r requirements.txt` in `/backend`.

## Pull Request Process
1. Ensure any install or build dependencies are removed before the end of the layer when doing a build.
2. Update the README.md with details of changes to the interface.
3. You may merge the Pull Request in once you have the sign-off of two other developers.

## Testing Requirements
We mandate 90% test coverage strictly. Run tests before submitting a PR!
