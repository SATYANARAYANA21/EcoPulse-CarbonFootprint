# Contributing to EcoPulse

Thank you for your interest in contributing to EcoPulse! We welcome contributions from the community to help make personal carbon tracking more accessible and impactful.

## How to Contribute

1. **Bug Reports**: If you find a bug, please open an issue with clear reproduction steps.
2. **Feature Requests**: We welcome ideas for new features or improvements.
3. **Pull Requests**:
   - Fork the repository.
   - Create a new branch for your feature or fix.
   - Ensure tests pass (`npm test` in frontend, `pytest` in backend).
   - Maintain high code coverage (90%+).
   - Follow the existing code style and accessibility standards.

## Development Setup

See the [README.md](../README.md) for local development setup instructions.

## Code of Conduct

Please be respectful and inclusive in all interactions related to this project.
